# Advanced Brainstorming Methods

Use these structured methods when a normal brainstorm stalls or becomes repetitive.

## SCAMPER

Seven ways to perturb an existing idea or problem frame.

- **Substitute**: Replace a method, model, assumption, dataset, or organism
- **Combine**: Merge two techniques, data sources, or viewpoints
- **Adapt**: Borrow a mechanism from another field
- **Modify**: Magnify, shrink, speed up, slow down, simplify, or exaggerate
- **Put to Another Use**: Reuse an existing method or failure mode elsewhere
- **Eliminate**: Remove a component, dependency, or assumption
- **Reverse**: Invert sequence, causality, or objective

## Six Thinking Hats

Force the conversation through six lenses:

- **White**: facts, data, missing information
- **Red**: intuition, discomfort, excitement
- **Black**: risks, weaknesses, failure modes
- **Yellow**: upside, value, optimistic path
- **Green**: creativity, alternatives, wild ideas
- **Blue**: process, synthesis, next steps

## Morphological Analysis

Break a problem into dimensions and enumerate combinations.

Example dimensions:

- system
- method
- measurement
- scale
- intervention

Then inspect the matrix for combinations nobody has explored.

## TRIZ-Style Contradiction Work

Useful when two goals appear incompatible.

Questions:

- What contradiction is currently treated as fundamental?
- Can segmentation, merging, local optimization, or prior action dissolve it?
- What would the ideal final result look like if the problem solved itself?

## Biomimicry

Translate the problem into a functional question and ask how nature solves it.

Process:

1. Define the function precisely
2. Search for natural systems doing something structurally similar
3. Abstract the strategy
4. Reapply the principle to the research problem

## Provocation

Use deliberately unreasonable statements to break fixation.

Examples:

- "The sample tells us what to measure."
- "The experiment runs itself."
- "The effect appears before the cause."

The point is not to believe the claim, but to see what new line of thought it unlocks.
